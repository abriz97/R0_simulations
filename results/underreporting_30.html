We try another underreporting experiment. In this case, I will just make use of Binomial Sampling to produce underreporting. I will not explicitely code for missed generations, even though these may arise naturally.

Again, I will set: \[ y_t \sim \text{Binom}(I_t, \pi). \]

Here, I want to apply the methods “as soon as possible”, according to the guidelines in the WebAppendix. In particular, for each of the “observation” time series, I want to obtain the shortest interval that contains at least 12 observations and is longer than 7 days. This means I will have to find the day of first observation, then if the weekly span contains at least 12 obs: GOOD, otherwise extend this until I reach 12. We can code a function that does exactly that:

find_window <- function(incidence, min_cases = 12, g_interval = 7, missed = 14){
  
  #cutoff missed days
  incidence <- incidence[-(1:missed),]
  
  ncols <- dim(incidence)[1]
  
  # find day of first infection
  start <- min(which(incidence$cases !=0))
  if(start == Inf){
    return(incidence)
  }
  
  end <- start + g_interval - 1
  #total cases in the window
  window_cases <- sum(incidence$cases[start:end])


  while( window_cases < min_cases & end < ncols ){
    
    end <- end + 1
    window_cases <- window_cases + incidence$cases[end]

  }
  
  return(incidence[start:end, ])
}

  # For every possible simulation
  for(c in 1:(ncol(df) - 1)){
    
    # analyse only if there are decent numbers of cases
    if (sum(df[,c]) > sqrt(1e+06)){
      
      # Get incidence
      incidence <- data.frame(cases = df[,c])
      incidence$cases <- rbinom(n = length(incidence$cases),
                                size = incidence$cases,
                                prob = pi)
      incidence$day <- 1:dim(incidence)[1]

      #== simple EpiEstim: unique window
